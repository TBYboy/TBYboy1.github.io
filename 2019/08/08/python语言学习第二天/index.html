二、内容复习：“列表及其操作”

1、列表简介

列表由一系列按特定顺序排列的元素组成，在python中用方括号（[ ]）来表示列表，并用逗号来分隔其中的元素。例如创建一个名字列表（后面的内容将围绕这个列表来展开）：

这个python的列表其实和c语言中的数组是相似的，它里面元素的索引（位置）是从零开始的，也就是说，names列表中的第一个元素‘TBY’的索引是 0。当然，访问列表中的元素也如同访问数组元素，指出元素的索引即可。比如我们要访问元素‘BF’，它的索引是 3，那么我们输入

print(names[3])

即可输出元素‘BF’，这里的names[3]即为元素‘BF’。

python同时为访问最后一个列表元素提供了一种特殊语法

print(names[-1]) = print(names[4]) = 元素‘ZYC’

同理，names[-2] = names列表中倒数第二个元素‘BF’。
 

2、列表操作

①. 对列表中元素进行修改、添加、删除操作 。
修改：
修改列表中元素非常简单，对要修改的元素位置进行赋值操作即可

names[1] = ‘YBY’

即把列表中2号元素‘LSB’修改为了‘YBY’。

添加：
对于在列表中添加元素，有以下俩种方法：
方法一：在列表末尾添加元素 ——– 借用函数append()

names.append(‘LYX’)

即把元素‘LYX‘添加到了列表末尾，此时names[-1] = ‘LYX’。

方法二：在列表中插入元素 ——– 借用函数insert()

names.insert(2,’LYX’)

即把元素’LYX’插入到了列表索引为2的位子上，原来位于这个位置上及以后的元素全部向后移动一个位置。

删除：
删除列表元素，有以下三种方法：
方法一：知道要删除的元素在列表中的位置 ——– 使用del语句

del names[1]

即把names中索引为 1 的元素’LSB’给删除了，此时names[1] = ‘GCY’。

方法二：将元素从列表中删除，但接着使用该元素的值 ——— 借用函数pop()

poped_name = names.pop(1)

这个语句的解释为删除names列表中索引为 1 的元素’LSB’，并将其值赋给变量poped_name。

注：“del”和”pop()”虽然都有删除功能，但”del”删除元素后元素再不可被使用，而”pop()”可以使用示例里的操作在删除元素的同时将其值赋给另一个变量，从而可以继续使用该元素。

方法三：根据值删除元素 ——– 借助函数remove()
在只知道删除元素的值但不知道其在列表中的位置时，用此方法。

names.remove(‘TBY’)

即把列表中的元素’TBY’给删除了。
 

②. 组织列表。
一、使用方法 sort() 对列表进行永久排序。
创建一个列表

elements = [‘c’,’d’,’b’,’a’]

若想让elements列表按字母表顺序排列则执行：

elements.sort()

即可得到有序列表elements = [‘a’,’b’,’c’,’d’]
而要倒序排列则执行：

elements.sort(reverse = true)

即可得到倒序列表elements = [‘d’,’c’,’b’,’a’]
