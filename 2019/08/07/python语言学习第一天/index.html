python学习     Day 1

———自学python，秃头的开始相信也是变强的开始。

一、准备工作

我选择的自学教材是【美】Eric Matthes 编著的书《Python编程 从入门到实践》

编译器是Microsoft Visual Studio 2017，个人觉得还行，可以去官网自行下载。
 

二、内容复习：“变量和简单数据类型”

本书的 第一章是环境搭建，所以就不再多叙，直接从第二章开始。
 
学习任何的语言，第一件要干的事当然都是输出简单的语句“hello world！”唠。听说编程界公认刚接触一门新语言时，首先运行输出这个会带来好运。

1、在python里，输出语句很简单:

2、python里的变量和c中的变量并无区别，其命名也是由字母、数字、下划线组成。

①. 变量命名只能由字母、数字、下划线组成，且不能以数字开头。
 
②. 变量名不可以包含空格，但可以用下划线来分割单词。
 
③. 不要将python关键词用作变量名，如”print”。

⑥. 数字处理：
在python中可以对整数进行加（+）、减（—）、乘（*）、除（/）操作，且不像c语言，python的数字操作可以整数、浮点数混合操作，例如：

3 * 0.1 = 0.3
3 / 2 = 1.5

注：结果包含的小数位数可能不确定。

除了基础的加减乘除，python还提供了幂次运算（**），并且可以通过括号改变运算的优先级。例如：

3 ** 3 = 27
( 2 + 3 ) ** 2 = 25
